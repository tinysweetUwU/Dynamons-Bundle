# Modded Bundle – Dynamons World

## Giới thiệu

Bundle này chứa các script đã được chỉnh sửa dành cho game Dynamons World, hỗ trợ nhiều phiên bản khác nhau trên Android và iOS.

## Cách sử dụng

1. Chọn script phù hợp với phiên bản game đang dùng
2. Inject hoặc load script vào game
3. Mở game và kiểm tra hoạt động

## Lưu ý

* Chỉ sử dụng cho mục đích học tập và nghiên cứu
* Có thể không hoạt động trên tất cả thiết bị hoặc phiên bản
* Khi game cập nhật, script có thể cần chỉnh sửa lại

## Ghi chú

* Hỗ trợ cả Android và iOS
* Bundle giúp test nhanh nhiều phiên bản khác nhau
